== Description ==

This is multilingual text widget, which works with qTranslate-X plugin.

You have to install qTranslate-X plugin before you use qText X Widget.

This plugins requires at least qTranslate-X 3.1 version.

After installing qText X Widget, you will have different input boxes for languages, which you have enabled with qTranslate-X.

== Installation ==

1. Download the plugin from [http://wordpress.org/extend/plugins/]
2. Extract content (1 folder containing 2 files) to your plugins directory
3. Activate the plugin from the 'Plugins' menu in Wordpress.
4. Enjoy :-)
